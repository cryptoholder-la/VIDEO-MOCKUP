### **Installation**

1. **Clone the repository**
```bash
git clone https://github.com/yourusername/motion-capture.git
cd motion-capture
```

2. **Install dependencies**
```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

3. **Set up environment variables**
```bash
cp .env.example .env
```

Add your environment variables:
```env
# Database
TURSO_CONNECTION_URL=your_turso_url
TURSO_AUTH_TOKEN=your_turso_token

# Authentication
BETTER_AUTH_SECRET=your_secret_key
BETTER_AUTH_URL=http://localhost:3000
```

```bash
npm run db:push
```

5. **Start the development server**
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

6. **Open your browser**
Navigate to [http://localhost:3000](http://localhost:3000)

## ğŸ“– Usage Guide

### **1. Upload Video**
1. Navigate to the Upload page
2. Drag and drop your video file or click to browse
3. Supported formats: MP4, MOV, AVI, WebM

- View interactive 3D model with applied motion
- Check motion statistics and timeline
- Review skeleton and facial rig structure

### **5. Export Data**

- Download includes full body motion + facial animation + lip sync
- Use in Blender, Maya, Unity, Unreal Engine, etc.
